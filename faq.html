怎樣應允第三方cookies在手機裏?

[安卓] 瀏覽器 - Google Chrome

  1. 在安卓手機或者平板電腦, 打開Chrome
  2. 往右上角, 按更多然後設定
  3. 按網頁設定然後Cookies
  4. 可以從哪裏設定開或者關
  5. 可選: 選擇應允第三方 cookies

[安卓] 瀏覽器 - Firefox

  1. 打開 Firefox
  2. 按目錄 (在某些設備的下方或者在網頁上的右上方)
  3. 按設定 (或者需要按更多先)
  4. 按隱私
  5. 按啟用

[IOS] 瀏覽器 - Safari

  1. 打開 Safari
  2. 選擇設定
  3. 在隱私&安全, 按阻止Cookies / 接受Cookies
  4. 一直應允或者只有指定網頁被選擇
 

怎樣清除緩存在手機裏?

[安卓] 瀏覽器 - Google Chrome

  1. 打開Chrome
  2. 按更多然後設定
  3. 在高級設定裏， 按隱私然後選擇清除瀏覽數據
  4. 在上面, 下拉的選單中 "清除資料"
  5. 選擇想要清除的時間
  6. 選擇想清除的選項
  7. 按清除資料

[安卓] 瀏覽器 - Firefox

  1. 打開 Firefox
  2. 按目錄 (在某些設備的下方或者在網頁上的右上方)
  3. 按設定 (或者需要先按更多)
  4. 按清除資料
  5. 選擇想要清除的資料然後按清除

[IOS] 瀏覽器 - Safari

  1. 打開 Safari
  2. 選擇設定
  3. 按清除歷史資料
  4. 按確定

 

推薦金優惠介紹

介紹好友即可獲得推薦金500，註冊時可填寫推薦碼(推薦人遊戲帳號)，並於註冊完成後24小時之內聯繫在線客服申請
 
  1. 提供被推薦人帳號、電話、lineID予在線客服進行申請
  2. 推薦人(舊會員)以及被推薦人(新會員)都須有存款1000以上之記錄(有領優惠之該筆存款不予計算)
  3. 每位相同註冊資料包含IP位置的新會員僅限被推薦一次，且需完成銀行綁定
  4. 舊會員可推薦多位好友，推薦優惠金無上限
  5. 該優惠進行彩票以及棋牌類型投注將不計算出款流水
  6. 若會員有重複註冊帳號惡意套取優惠或其他非正當盈利之行為，將扣除優惠以及所有不當盈利
  7. 出款流水僅需一倍即可
  8. 若同時推薦多位好友，每位都需有存1000以上之紀錄(有領優惠之該筆存款不予計算)
